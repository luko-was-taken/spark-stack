# spark-stack
## Getting Started

### Prerequisites
* docker
* docker-compose

### Installation

1. Clone the repo and cd into it 
   ```sh
   git clone https://github.com/luko-was-taken/spark-stack.git && cd spark-stack
   ```
2. Build base images
   ```sh
   ./build.sh
   ```
3. Launch the docker compose file
   ```js
   docker-compose up -d
   ```
